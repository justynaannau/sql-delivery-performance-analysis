# SQL Delivery Performance Analysis

## Project Overview
SQL analysis project focused on e-commerce delivery performance and operational efficiency.

The analysis was performed using SQL Server Management Studio (SSMS) on a Brazilian e-commerce dataset.

## Objectives
- analyze delivery delays
- monitor monthly order trends
- identify operational risk periods
- evaluate delivery performance KPIs
- check data quality issues

## Tools Used
- SQL Server
- SQL Server Management Studio (SSMS)
- GitHub

## Analysis Included
- duplicate order detection
- monthly orders trend
- order status analysis
- average delivery time
- delivery status categorization
- monthly delay rate
- top delayed months
- missing delivery date analysis
- average delay days
- monthly growth analysis

## Dataset
Brazilian E-commerce Public Dataset (Olist)
